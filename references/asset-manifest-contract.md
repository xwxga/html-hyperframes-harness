# Asset Manifest Contract

The asset manifest records real source assets. It prevents the board from drifting into invented visuals.

## Required Fields

- Asset ID
- Path or URL
- Type
- Source
- Rights status
- Visual role
- Target frames
- Risk

## Rules

- Do not invent assets.
- Every must-use asset must appear in the Director Workbench Asset Wall.
- Private, licensed, or unverified assets must be marked before publishing.
- Final HyperFrames source should preserve asset IDs as metadata when practical.
