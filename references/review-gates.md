# Review Gates

Use these gates before final HyperFrames generation.

## Director Workbench Gate

- Overview starts with real visual evidence.
- Asset Wall shows all must-use assets.
- Critical Frame Mockups show real frame states.
- Frames view exposes every planned final frame.
- Frames Inspector declares HyperFrames default critical-frame style.
- Motion Timeline explains motion intent without implementation code.

## Handoff Gate

- Accepted review changes are synced back to Markdown.
- `04_render_plan.md` is marked ready.
- Final source mapping is clear enough for the next agent.
- No retired examples, old comment-system files, secrets, or large unplanned media are present.
